Ce projet a pour but de créer la recette optimale pour un fondant au chocolat du tonnerre.

Le projet se compose de 2 fichiers :

- ingredients.txt
- preparation.txt

Contributions :

Vous pouvez contribuer à cette recette en respectant ces termes :

- Créer et travailler sur une nouvelle branche du projet
- Créer des commits pour enregistrer vos modifications
- Pousser les commits sur Github
- Créer une Pull request

Merci et régalez vous !
